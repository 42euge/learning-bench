# Machine Translation from One Book (2023)

#paper #literature #learning

**Authors:** Garrett Tanzer, Mirac Suzgun, Eline Visser, Dan Jurafsky, Luke Melas-Kyriazi
**Source:** arXiv:2309.16575

## Key Contribution
Introduces MTOB (Machine Translation from One Book), a benchmark that tests whether LLMs can learn to translate Kalamang -- an extremely low-resource language with fewer than 200 speakers and virtually no web presence -- using only a grammar book, wordlist, and a small parallel corpus provided in-context.

## Problem Addressed
Standard machine translation relies on large parallel corpora. Humans, however, can learn new languages from pedagogical materials (grammar books, dictionaries). No benchmark existed to test whether LLMs can similarly acquire a genuinely novel language from structured reference materials, as opposed to recalling training data.

## Method
- Curates a dataset from field linguistics materials for Kalamang: a grammar book (~300 pages), a bilingual wordlist, and a small set of parallel sentences.
- Frames the task as L2-style language learning: given these reference materials in-context, translate between Kalamang and English.
- Evaluates frontier LLMs (GPT-4, PaLM 2) using various prompting strategies with different subsets of reference materials.
- Compares LLM performance against a human baseline (a linguist who learned Kalamang from the same materials).

## Key Results
- Best LLM performance: 44.7 chrF (Kalamang-to-English) and 45.8 chrF (English-to-Kalamang).
- Human baseline: 51.6 chrF (Kal-to-Eng) and 57.0 chrF (Eng-to-Kal).
- LLMs are promising but fall meaningfully short of human performance, especially on production (Eng-to-Kal).
- The grammar book is more useful than the wordlist alone, showing models can extract structural knowledge.

## Limitations
- Single target language (Kalamang); generalizability to other low-resource languages is untested.
- Context window limits constrain how much reference material can be provided.
- chrF metric may not capture all dimensions of translation quality.
- Does not test iterative learning or retention across sessions.

## Relevance to Our Research
Exemplary benchmark for the Learning track. MTOB isolates genuine in-context learning by using a language absent from training data, making memorization impossible. The gap between human and LLM performance provides clear discriminatory signal. Its methodology of using structured pedagogical materials as learning input is directly applicable to designing tasks that test knowledge acquisition.
