Required · Linked as project link · Private until deadline

Benchmark Structure

The benchmark consists of 12 tasks organized into 6 cognitive sub-abilities. Each task is a self-contained Kaggle Benchmark notebook that uses the kaggle_benchmarks SDK to evaluate models via the LLM proxy.

Task Notebooks

Each notebook lives at tasks/{sub-ability}/{task-name}/{task-name}.ipynb and follows the pre/study/post paradigm.
