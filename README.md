# Learning Benchmark

Kaggle benchmark for the **Learning** track of the Google DeepMind x Kaggle AGI Hackathon.

## Structure

- `src/` — benchmark code
- `data/` — dataset files
- `results/` — model evaluation outputs
- `notebooks/` — exploration & analysis

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # then fill in your MODEL_PROXY_API_KEY
```
